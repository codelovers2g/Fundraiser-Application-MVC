People Export Filter:

It is a type of filter where we get a list of people on the basis of different options selected. For example We can filter a people on the basis of:

1. Gender
2. Maritial Status
3. Title
4. Deceased
5. City
6. State
7. Zip
8. Country etc.

Along with these we can add additional filter as per the financial record:

1. We can add Financial Columns for which we can set 
	a. Amount( min or max)
	b. Date(From or to)
	c. Type
2. We can also add Group Columns(With which Group a person associated according to a organization) for which we can set
	a. Group Type
	b. Group
