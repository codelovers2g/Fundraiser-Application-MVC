Template System : It is used to create a on the fly template for email and as well as for a document. It is designed as an online template creator in which one can type 
content manually and can also use  merge fields for replace their respective values dynamically. It is using all the formats for text.

Challanging Parts:

1. Replace values dynamically 
2. Also there is additional text functionality which is used to add additional text at a time in template.
3. Attach mutiple files with email at a time
4. Add an image with document and show it as a background of it.
