Kendo Filter :

Kendo filter is a type of filter in which we can filter records in a Group Format. Here we can Group the records according to a different options for person. For example:

In this we can filter a person according to its personal and financial records.We can get a records according to financial options selected after that we can group individual record
according to a personal information

Challenging Part: There is a two parts Grouped and detail 

1. Grouped: Initially all data is grouped as a count. We can group individual record depends upon different fields.
2. Detailed part show all the records with detail but we can also group it by dragging a column on top.
